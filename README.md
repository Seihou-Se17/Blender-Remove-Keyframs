# Blender-Remove-Keyframs
Blender add-on for delete keyframes from the selected object.

選択したオブジェクトからキーフレームを削除するためのBlnederアドオンです。


This is a very simple add-on.

Select one or more objects, press the button that says Delete all keyframes.

このアドオンはとてもシンプルです。
オブジェクトを1つ以上選択し、全てのキーフレームを削除するというボタンを押します。


In the future, we plan to make it possible to specify a more precise range and delete keyframes.

今後、より細かい範囲を指定し、キーフレームを削除することができるようにする予定です。

### Supported Versions
Blender Version 5.0 or higher
